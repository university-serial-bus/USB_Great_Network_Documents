## 1. プライベート IPv4 アドレッシング メモ

### 1.1. 全体 IPv4 アドレス (全体 /13) メモ
**10.168.0.0/13 (10.168.0.0 ～ 10.175.255.255)**  
サブネットマスクは 255.248.0.0

「こういうけったいな大学間連携の閉域学生プライベートネット遊び」のための /13 を概念的に定義して合意。これは 10.x.0.0/13。できるだけ相互の既知のネットワークとかぶらないほうがよい。(将来 NAT などでルーティングしたりするかも知れないから)

x は、覚えやすいように、168 (イロハ)


### 1.2. 上記のうち一部の AoE セグメント (/18) ・・・ これが実在セグメントである !!!
**AoE セグメント: 10.168.0.0/18 (10.168.0.0 ～ 10.168.63.255)**  
サブネットマスクは 255.255.192.0

DefGW (NAT でインターネットに出る): 10.168.63.254  
一応のユーザ用 DNS サーバー: 10.168.63.254

DefGW と DNS は、当面、か●●もんさんのルータで実施いただく。  
なお、AoE を行なう場合、DefGW や DNS は、落ちていてもよい。

IPv6 は、あっても、なくてもよい。AoE には影響しない。

### 1.3. 上記のアドレス範囲のうち一部を各拠点で DHCP プールとしてそれぞれ自律配布 (おおむね /24)

将来 慶應 SFC, 筑波大学以外に第三、第四がきたら、増やしていく。最大 64 組織。

- **SFC 第 1 DHCP (デルタ棟 2F: かわえもんネット起点): 10.168.1.101 ～ 10.168.1.199 を配布。**何か固定で置きたい場合は 10.168.1.1 ～ 100 を利用。
- **筑波 第 1 DHCP (Ultra-Coins): 10.168.2.101 ～ 10.168.2.199 を配布。**何か固定で置きたい場合は 10.168.2.1 ～ 100 を利用。
- **筑波 第 2 DHCP (WORD 部屋): 10.168.3.101 ～ 10.168.3.199 を配布。**何か固定で置きたい場合は 10.168.3.1 ～ 100 を利用。
- **筑波 第 3 DHCP (AC 部屋): 10.168.4.101 ～ 10.168.4.199 を配布。**何か固定で置きたい場合は 10.168.4.1 ～ 100 を利用。
- **IPA (サイバー技術研究室): 10.168.5.101 ～ 10.168.5.199 を配布。**何か固定で置きたい場合は 10.168.5.1 ～ 100 を利用。

###  1.4. (各拠点オプショナル) それぞれの室で、次の設定で WiFi を吐く。AoE マシンのスムーズな接続用。設定を合わせることで、ある拠点から別の拠点に AoE マシンを持って行ってもパッと接続できて、ゲームがすぐに開始できる。

・ 5 GHz (11ac/n/a) <-- こちらを推奨  
SSID: AoE_WiFi_5G_WPA  
無線の認証: WPA/WPA2 mixed PSK  
暗号化: TKIP/AES mixed  
WPA-PSK: microsoft  
倍速モード 帯域: 346.7 Mbps (20 MHz)  
無線チャンネル: 36 (可能であれば)   

・ 2.4 GHz (11n/g/b) <-- 5GHz 帯が接続できないマシンのみ  
SSID: AoE_WiFi_2G_WPA  
無線の認証: WPA/WPA2 mixed PSK  
暗号化: TKIP/AES mixed  
WPA-PSK: microsoft  
倍速モード 帯域: 20 MHz  
無線チャンネル: 1 (可能であれば)
